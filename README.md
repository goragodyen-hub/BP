# 🩺 บันทึกความดัน (Blood Pressure Tracker)

แอปพลิเคชันสำหรับบันทึกและติดตามผลความดันโลหิตที่บ้าน (Home Blood Pressure Monitoring) ถูกออกแบบมาให้ใช้งานง่ายบนสมาร์ทโฟนและคอมพิวเตอร์ ข้อมูลทั้งหมดจะถูกส่งไปเก็บไว้ใน **Google Sheets** ของคุณโดยตรง เพื่อความปลอดภัยและความเป็นส่วนตัวของข้อมูลสุขภาพ

![Blood Pressure Tracker App](public/logo.png)

## ✨ ฟีเจอร์หลัก (Features)

- **📝 บันทึกค่าความดัน:** รองรับการบันทึกค่าความดัน 2 ช่วงเวลา (เช้า/ก่อนนอน) และ 2 ครั้งต่อช่วงเวลา (ห่างกัน 1 นาที) ตามมาตรฐานทางการแพทย์
- **📊 สรุปรายงานและกราฟ:** แสดงผลค่าเฉลี่ยความดันตัวบน (SYS) และตัวล่าง (DIA) พร้อมกราฟเส้นแสดงแนวโน้มสุขภาพของคุณ
- **🖨️ พิมพ์ประวัติเป็น PDF:** กดปุ่มเดียวเพื่อแปลงประวัติการวัดความดันเป็นไฟล์ PDF ขาวดำที่อ่านง่าย พร้อมนำไปปรินต์ส่งให้แพทย์ได้ทันที
- **🚦 ประเมินระดับความดันอัตโนมัติ:** มีระบบป้ายกำกับสี (เขียว/เหลือง/แดง) แจ้งเตือนเมื่อค่าความดันสูงกว่าเกณฑ์มาตรฐาน
- **📱 รองรับ PWA:** สามารถติดตั้งแอปพลิเคชันลงบนหน้าจอโฮมของมือถือเพื่อการใช้งานที่สะดวกเหมือนแอปพลิเคชันทั่วไป

## 🚀 วิธีการติดตั้งและใช้งาน

ระบบนี้แบ่งออกเป็น 2 ส่วน คือ **หน้าเว็บแอปพลิเคชัน (Frontend)** และ **ระบบฐานข้อมูล (Backend ด้วย Google Sheets)**

### ส่วนที่ 1: ตั้งค่า Google Sheets (ฐานข้อมูล)
1. เปิด [Google Sheets](https://docs.google.com/spreadsheets/) สร้างเอกสารใหม่
2. ไปที่เมนู **ส่วนขยาย (Extensions)** > **Apps Script**
3. ลบโค้ดเดิมออกให้หมด แล้วคัดลอกโค้ดทั้งหมดจากไฟล์ `apps-script.js` ไปวาง
4. กดปุ่ม **"นำไปใช้งาน" (Deploy)** ที่มุมขวาบน > เลือก **"การนำไปใช้งานชุดใหม่" (New deployment)**
5. กดรูปเฟือง ⚙️ ทางซ้าย เลือกประเภทเป็น **"เว็บแอป" (Web app)**
6. ในช่อง "ผู้ที่มีสิทธิ์เข้าถึง" (Who has access) ให้เลือกเป็น **"ทุกคน" (Anyone)**
7. กด Deploy และให้สิทธิ์การเข้าถึงบัญชี (หากมีแจ้งเตือน)
8. **คัดลอก "URL ของเว็บแอป"** ที่ขึ้นต้นด้วย `https://script.google.com/...` เก็บไว้

### ส่วนที่ 2: ตั้งค่าหน้าเว็บแอปพลิเคชัน (Frontend)
1. นำ URL ที่คัดลอกมา ไปวางในไฟล์ `src/App.jsx` ที่บรรทัดตัวแปร `SCRIPT_URL`
   ```javascript
   const SCRIPT_URL = 'https://script.google.com/macros/s/.../exec';
   ```
2. หากต้องการรันระบบในเครื่องคอมพิวเตอร์ของคุณ ให้เปิด Terminal ในโฟลเดอร์โปรเจกต์และพิมพ์คำสั่ง:
   ```bash
   npm install
   npm run dev
   ```
3. เปิดเบราว์เซอร์ไปที่ `http://localhost:5173` เพื่อเริ่มใช้งาน!

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

- **Frontend:** React, Vite
- **Styling:** Vanilla CSS (Glassmorphism UI)
- **Charts:** Recharts
- **Icons:** Lucide React
- **Backend / Database:** Google Apps Script + Google Sheets
- **Deployment (CI/CD):** GitHub Actions (สำหรับ Deploy ขึ้น GitHub Pages)
